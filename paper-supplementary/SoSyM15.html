Supplementary Material for SoSyM 15 Paper

This page provides supplementary material for the paper WESSBAS: Extraction of Probabilistic Workload Specifications for Load Testing and Performance Prediction - A Model-Driven Approach for Session-Based Application Systems.

The material has been updated for the revised version submitted in February 2016.

Contents

Software

The following WESSBAS software relevant to the paper is available via ​https://github.com/Wessbas/:.

  • ​wessbas.commons --- Common functionality for the other WESSBAS projects
  • ​wessbas.dsl --- WESSBAS DSL
  • ​wessbas.sessionExtraction --- Generates a session log from a Kieker monitoring log; analysis program for session logs
  • ​wessbas.behaviorModelExtractor --- WESSBAS User Behavior Model Extractor (includes the clustering)
  • ​wessbas.dslModelGenerator --- WESSBAS Model Extractor
  • ​wessbas.markov4jmeter the WESSBAS (Markov4JMeter) extension for JMeter, including extensions for think times
  • ​wessbas.testPlanGenerator --- WESSBAS Test Plan Generator
  • ​wessbas.wessbas2pcm --- Transformation from WESSBAS-DSL instances to PCM
  • ​wessbas.kiekerExtensions --- Additional WESSBAS-specific Kieker records and Servlet probes

The WESSBAS UI is available as a password-protected (password: wessbasui) ZIP file:

  • ​https://dl.dropboxusercontent.com/u/81621779/wessbas.ui.zip --- WESSBAS GUI (license confirmation pending, i.e., not on GitHub, yet)
